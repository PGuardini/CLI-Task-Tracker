# CLI-Task-Tracker
CLI Task Tracker is a simple CRUD exercise built in Python that store the user's tasks in a JSON file.

## Running the app

- To run this simple app you must have Python 3 installed
- Run in command line the follow command in the repository directory:
  > py task_manager.py
- Once runned, a file named 'tasks.json' is created
- Now you can use the task tracker to track and manage your tasks

- Have fun! :)
